# Flutter Chat Demo

A Group chat Flutter project .

This is a chat demo application using (flutter and firebase), it contain:
- One to one chat
- Group chat
- Record voice
- Pick multiple images
- Using (Google sign in - cloud firestore - Firebase Functions - Cloud messaging)
